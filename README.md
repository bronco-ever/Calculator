# CSS Calculator (Vanilla HTML/CSS + Inline `onclick`)

A clean, responsive calculator built with **vanilla HTML and CSS**, using simple **inline `onclick` events** for input handling.  
This project focuses on layout, styling, and basic calculator behavior without frameworks.
